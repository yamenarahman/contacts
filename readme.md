# Contacts

### Tech used:

- Laravel 5.7+
- Vuejs 2.5+
- Bootstrap 4
- Axios library
- Laravel mix (yarn + webpack).

### Installation:

1. clone the repo.
2. run `composer install`.
3. run `cp .env.example .env` and set database credentials in .env file.
4. run `php artisan key:generate && php artisan migrate`.
5. serve the project `php artisan serve`.
